## Full Description

```text
GPS Spoofer is an Android utility for testing how location-aware apps behave when device coordinates change.

Set a single mock location, scrub along a route, or simulate movement automatically at a chosen speed. The app is built for QA, demos, field simulations, and development workflows where you need controlled GPS behavior on a real Android device.

Features:
- Mock a single location or simulate movement along a route
- Load routes from encoded polylines or Google Routes JSON
- Add, drag, rename, reorder, save, and reload custom waypoints
- Search places with device geocoding and jump the map quickly
- Scrub route progress manually or use play and pause with speed control
- Keep spoofing active in background mode with a persistent notification
- Run setup checks for permissions, developer options, and mock app status
- Choose dark mode behavior for the app UI and the map

Requirements:
- A compatible Android device
- Developer Options enabled
- This app selected as the mock location app
- Location permission granted

Notes:
- Place search depends on geocoder support available on the device
- This app is intended for testing, QA, demo, and development use

Use GPS Spoofer only where you have permission to simulate location data.
```
