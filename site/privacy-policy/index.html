Privacy Policy

GPS Spoofer by VoxTourAI

GPS Spoofer is an Android utility used to test GPS and mock-location behavior in location-aware apps, including apps such as voxtour.ai. It is intended for testing, QA, demo, and development workflows, not for advertising, profiling, or consumer tracking.

Effective Date March 14, 2026
Developer VoxTourAI
Privacy Contact admin@voxtour.ai

What This App Does

GPS Spoofer lets a user set a mock device location or simulate movement along a route on Android. This can help teams test how location-aware apps behave when GPS coordinates change.

When mock location is enabled, other apps on the same Android device may receive the mocked location through Android location services. That is the intended function of this utility.

Data We Access

  • Device location, including precise or approximate location, to show the current location, position the map, and support mock location testing.
  • User-entered route data, waypoint names, and search text when you choose to paste routes, save routes, or search for a place.
  • Basic local app settings such as terms acceptance, startup prompt state, and saved custom routes.

How We Use Data

  • To render the in-app map and help the user inspect test routes.
  • To allow the user to set or simulate a mock GPS location on the Android device.
  • To save routes and settings locally on the device for convenience.
  • To let the user search for addresses or places from within the app.

Data We Do Not Sell or Use for Advertising

VoxTourAI does not sell personal information from this app. This app is not designed as an advertising, data-broker, or consumer-tracking product.

Where Data Goes

  • Saved routes and app preferences are stored locally on your device.
  • The app uses Google Maps functionality for map display, which may involve requests handled by Google.
  • If you use in-app address search, the query may be processed by Android or service providers available on the device to resolve a location result.
  • Other apps on the device may observe the mocked location while mock location is active, because Android exposes that location through the system location stack.

We do not describe any separate VoxTourAI backend collection in this app for account login, ad targeting, analytics profiling, or cloud sync.

Retention and Deletion

  • Saved routes and settings remain on the device until you delete them in the app, clear app storage, or uninstall the app.
  • Mock location values are temporary test state and are replaced or removed when you stop using the feature.
  • VoxTourAI does not describe server-side retention of this app's route or location data because this app stores its route/settings state locally on-device.

Security

We rely on Android app sandboxing and platform permissions to help protect locally stored data. You should still use this app only on devices and in environments you trust, especially when mock location could affect other installed apps.

Children

This app is a developer and tester utility and is not directed to children.
